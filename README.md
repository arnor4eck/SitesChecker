[![Java](https://img.shields.io/badge/Java-21-orange.svg)](https://openjdk.java.net/)
[![JavaFX](https://img.shields.io/badge/JavaFX-21-blue.svg)](https://openjfx.io/)
[![Gradle](https://img.shields.io/badge/Gradle-8.14-green.svg)](https://gradle.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://mit-license.org/)

Приложение для периодического мониторинга доступности сайтов и отслеживания изменений их контента с графическим интерфейсом на JavaFX.

---

## 📖 Описание

**SitesChecker** позволяет:
- ✅ Добавлять сайты на мониторинг с настраиваемым периодом проверки
- ✅ Отслеживать HTTP-код ответа каждого сайта
- ✅ Обнаруживать изменения контента через хеширование
- ✅ Просматривать историю проверок
- ✅ Получать уведомления об ошибках доступа

## 📚 Как использовать

### 1. Запуск приложения
После запуска откроется главное окно с тёмной темой.

### 2. Создание задачи мониторинга
1. Нажмите кнопку **«+»** или **«Создать задачу»** в верхней панели
2. Заполните форму:
   - **Название** — имя задачи (например, "Google Homepage")
   - **URL** — адрес сайта (например, `https://google.com`)
   - **Период** — интервал проверки
   - **Единица** — секунды, минуты или часы
3. Нажмите **«Сохранить»**

### 3. Просмотр статистики
- На главной панели отображаются все задачи мониторинга
- При выборе задачи в показывается история проверок:
  - Время проверки
  - HTTP-код ответа
  - Изменился ли контент

### 4. Управление задачами
- **Удалить задачу** — выделите задачу и нажмите кнопку удаления
- **Остановить/Запустить** — кнопка в верхней панели приостанавливает все проверки

### 5. Логи
В нижней части окна отображаются логи приложения:
- ✅ Информация о успешных проверках
- ⚠️ Предупреждения о проблемах
- ❌ Ошибки подключения

## 💾 База данных

Приложение использует SQLite для хранения данных. Файл БД создаётся автоматически:

- **Имя файла:** `checker-app.db`
- **Расположение:** корень проекта (или директория запуска)

### Таблицы

#### `monitoring_task`
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Первичный ключ |
| name | TEXT | Название задачи |
| url | TEXT | URL сайта |
| next_check_time | TEXT | Время следующей проверки |
| period | INTEGER | Период проверки |
| unit | TEXT | Единица измерения (Секунда/Минута/Час) |

#### `site_statistics`
| Поле | Тип | Описание                      |
|------|-----|-------------------------------|
| id | INTEGER | Первичный ключ                |
| check_time | TEXT | Время проверки                |
| http_code | INTEGER | HTTP-код ответа               |
| hash | TEXT | Хеш контента                  |
| is_same_hash | BOOLEAN | Совпадает ли хеш с предыдущим |
| monitoring_task_id | INTEGER | Внешний ключ на task          |
